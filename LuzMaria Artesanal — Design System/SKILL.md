LuzMaria Artesanal is chiaroscuro and devotional: candlelight against deep dark, aged
gold as reflected light (sparing — never wallpaper), wine and ivory accents, elegant
serif display type. Baroque ornament is a pointed accent, never a repeated background.
Conversion happens on WhatsApp; keep CTA and form zones clean and high-contrast. Voice is
warm, reverent, first-person-plural for the brand — never salesy. Brazilian Portuguese.

If creating visual artifacts (slides, mocks, throwaway prototypes), copy assets out and
create static HTML files for the user to view. If working on production code, copy assets
and read the rules here to become an expert in designing with this brand.

If the user invokes this skill without other guidance, ask them what they want to build or
design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_
production code, depending on the need.

Note: this system ships with substitutions flagged in `readme.md` (fonts, logo,
photography). Honor those notes and ask the user for real brand assets when relevant.
