**Button** — the primary action surface; clean and high-contrast, no ornamentation (keep flourishes away from anything functional).

```jsx
<Button variant="primary" size="lg">Quero fazer minha encomenda</Button>
```

Variants: `primary` (aged-gold fill on ink — the conversion CTA), `secondary` (gold hairline outline), `ghost` (text only, for nav/tertiary), `onIvory` (dark fill for use on ivory/inverted blocks). Sizes `sm | md | lg`. Supports `iconLeft` / `iconRight`, `fullWidth`, `disabled`. Use ONE primary CTA per view.
