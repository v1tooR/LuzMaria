**Tag** and **Badge** — small labels.

```jsx
<Tag tone="gold">Personalização com nome</Tag>
<Badge tone="gold">Feito à mão</Badge>
```

`Tag` is a pill for personalization options / occasions (tones: `gold | wine | muted`). `Badge` is a tiny uppercase marker for status (tones: `gold | wine | outline`). Both reference brand tokens; use sparingly.
